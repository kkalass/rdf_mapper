Creates a reader for fluent access to resource properties.

The resource reader provides a convenient API for reading properties from an RDF subject, handling common patterns like retrieving single values, collections, or related objects. It encapsulates the complexity of traversing the RDF graph and extracting structured data.

The reader pattern simplifies the process of accessing multiple properties of a subject, especially when dealing with complex nested structures.

Example usage:

final reader = context.reader(subject);
final name = reader.require<String>(foaf.name);
final age = reader.require<int>(foaf.age);
final friends = reader.getList<Person>(foaf.knows);

Creates a reader for fluent access to resource properties.

subject is the subject term (IRI or blank node) to read properties from.

Returns a ResourceReader instance for fluent property access.
