Adds multiple child nodes to this node.

Use this method to add a collection of objects as related nodes. Each object in the collection will be serialized to its own set of RDF triples and linked to the current subject via the specified predicate. Note that in RDF terms this corresponds to multi-value properties, not to collections or lists.

Example:

builder.childResources(foaf.knows, friends);
builder.childResources(schema.author, authors);

@param predicate The predicate for the relationships @param values The collection of child node values @param serializer Optional serializer for the child node type @return This builder for method chaining
