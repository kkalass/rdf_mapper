Adds multiple child nodes extracted from a source object.

This method is useful when you need to extract multiple complex objects from a parent object and serialize each as a separate linked node.

Example:

// Extracts all chapters from a book and adds them as structured content
builder.childNodes(schema.hasPart, (book) => book.chapters, currentBook);

@param predicate The predicate for the relationships @param toIterable A function that extracts the collection of objects from the source @param instance The source object to extract values from @param serializer Optional serializer for the child node type @return This builder for method chaining
