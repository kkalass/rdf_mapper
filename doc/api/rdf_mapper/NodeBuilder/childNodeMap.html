Adds a map of key-value pairs as child nodes.

This method is useful for serializing dictionary-like structures where both the keys and values need to be serialized as part of the RDF graph.

Example:

// Serializes a metadata dictionary as linked nodes
builder.childNodeMap(
  schema.additionalProperty,
  metadata,
  MetadataEntrySerializer(),
);

@param predicate The predicate for the relationships @param instance The map to serialize @param entrySerializer The serializer for map entries @return This builder for method chaining
