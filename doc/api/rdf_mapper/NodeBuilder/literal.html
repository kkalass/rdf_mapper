Adds a literal property to the node.

Use this method to add a property with a literal value (strings, numbers, dates, etc.) to the current subject. The value will be serialized to an RDF literal term using the appropriate serializer.

Example:

builder.literal(dc.title, 'The Title');
builder.literal(foaf.age, 30);

@param predicate The predicate IRI for the property @param value The literal value to add @param serializer Optional custom serializer for the value type @return This builder for method chaining
