Deserializes an object of type T from an RDF graph.

This method finds and deserializes a subject of the specified type in the graph. It will attempt to find a subject with an rdf:type triple matching a registered deserializer for type T.

Note that this method expects the graph to contain exactly one deserializable subject of the specified type. If multiple subjects could match, use deserializeBySubject or deserializeAll instead.

Example:

final person = graphOperations.deserialize<Person>(graph);

@param graph The RDF graph to deserialize from @param register Optional callback to register temporary mappers @return The deserialized object of type T @throws TooManyDeserializableSubjectsException if multiple subjects can be deserialized @throws NoDeserializableSubjectsException if no deserializable subject is found
