CircularRdfListException

CircularRdfListException class

Exception thrown when a circular reference is detected in an RDF List structure.

RDF Lists should form linear chains ending with rdf:nil. Circular references indicate malformed data that would cause infinite loops during traversal.

This exception extends InvalidRdfListStructureException to maintain the exception hierarchy while providing specific information about cycle detection.
