Retrieves values from an RDF collection, using a provided collection deserializer.

collectionDeserializerFactory is a factory function that provides an instance of UnifiedResourceDeserializer for the specific collection type C and item type T.

For example, to deserialize an rdf:List into a List<Chapter>: reader.requireCollection<List<Chapter>, Chapter>(SchemaBook.hasPart, RdfListCollectionDeserializer.new)
