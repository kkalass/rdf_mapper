FragmentIriTermMapper

FragmentIriTermMapper class final

Bidirectional mapper for IRI fragments (the part after #).

This mapper extracts the fragment from absolute IRIs during deserialization and appends fragments to a base IRI during serialization. Useful for document-relative references, anchor links, or identifier systems based on fragments.

Examples: With base IRI "http://example.org/document":

  • Serialization: "section1" → const IriTerm("http://example.org/document#section1")
  • Deserialization: const IriTerm("http://example.org/document#section1") → "section1"

For IRIs without fragments, deserialization returns an empty string.
