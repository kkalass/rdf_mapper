UnorderedItemsDeserializerMixin

UnorderedItemsDeserializerMixin<T> class abstract mixin

Mixin providing deserialization functionality for unordered collections.

This mixin encapsulates the common logic for converting RDF objects back to collections using the multi-objects approach. It deserializes each RDF object and collects the results into a collection.

Multi-Objects Deserialization: Processes multiple RDF objects with the same predicate:

<subject> <predicate> "first element" .
<subject> <predicate> "second element" .
<subject> <predicate> "third element" .

Type Parameters:

  • T: The element type within the collection
